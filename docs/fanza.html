FANZA销量榜 TOP榜单分析

2023/01/20 发表于 jinjier.art


背景介绍


FANZA销量榜每年初的时候都会公布上一年的累计销量榜，即FANZA销量年榜。

本文将对照JavLibrary & JavDB TOP榜单分析，对FANZA影片年榜的历史累计数据进行统计分析。


榜单分析


本文数据抓取自DMM网站，文中涉及排名的部分均按进入榜单的总次数进行排序。

可以使用以下SQL语句在JAV SQL 查询器中查询FANZA影片年榜的历史数据：

SELECT number, icon_url, SUBSTR(name, 0, 40), note
FROM ranks
WHERE note LIKE '影片榜%全年'
ORDER BY date DESC, number
执行SQL


影片数量


  • 影片总数：626
  • 蓝光总数：169
  • 共演总数：33

FANZA只收录有码影片，因此上榜的影片都是有码影片。其中蓝光碟169部。有33部由多名女优共演。


年代分布



上图是FANZA影片年榜的年代分布。数据始自2007年，也包括了刚刚公布的2022年榜。

理论上FANZA影片年榜每年都应该是100部，但因为各种原因（如女优利用五年下架权利，或FANZA刻意隐藏部分数据等），早期的数据缺失很大，如2007-2013年我们抓取到的都只有个位数。2016年后数据开始增多，2020年后则都是完整数据。

从以上年代分布来看，虽然这份榜单起始自2007年，但主要反映的是2016年后的业界趋势。


片商排名



上图是按入榜影片数量排名前十的制作商，与JavLibrary & JavDB TOP榜单分析中的排名差异很大：

  • S1高居榜首，而且基本呈碾压之势。
  • 在JavLibrary和JavDB均排名第一的IDEA POCKET仅排名第四，且入榜数只有S1的四分之一。国内最受欢迎的IDEA POCKET在日本确实只是二流片商。
  • 客场作战的SOD杀入了前二，而它在国内榜单不算顶级（JavLibrary第六、JavDB第五）。
  • Madonna和Attackers都表现平平，FALENO和MUTEKI则冲到了第五、第六，这都与国内反差很大。
  • 在国内颇受欢迎的蚊香社(PRESTIGE)表现惨不忍睹，客场作战没有一部影片入围。

比较中日两国的榜单，不难看出两国审美的巨大差异。这也是我们要把片商排名放在一开始的原因，它也为本文接下来的排名及分析定下了基调。

为节省篇幅，片商完整排名可在附录中查看。


类别排名



上图是入榜影片的类别标签数量排名前十。我们可以与Javlibrary & JavDB的TOP10类别作一对比：

  • 「巨乳」在三个榜单都排名第一。具体到各个榜单，巨乳在FANZA的比例为52%(326/626)，JavLibrary为27.4%（137/500)，JavDB为30%(75/250)。此外，「乳交」也进入了FANZA的TOP10，而这个类别在JavLibrary和JavDB的排名都不高。
  • 「首次亮相」在FANZA极受欢迎，占比为24.4%(153/626)。而在JavLibrary仅为0.8%(4/500)，在JavDB更是为0。
  • 「艺人」在FANZA也极受欢迎，占比21.6%(135/626)。而在JavLibrary仅为1.6%(8/500)，JavDB则没有收录这个类别。
  • 「中出」在FANZA占比为12%(75/626)。在JavLibrary占比18.2%(91/500)，JavDB占比29.6(74/250)。

通过以上的词频分析，结合TF-IDF算法，我们大致可以得出以下结论：

  1. 在日本市场最受欢迎的影片类型为新人、艺人、巨乳等，以肉片为主。
  2. 在国内比较受欢迎的是丝袜、美腿、中出等，国内观众更偏爱剧情片。

总结


本文通过对FANZA影片销量年榜的历史数据进行统计分析，并对比JavLibrary & JavDB TOP榜单分析，可以看出中日两国观众在审美及喜好上的巨大差异。

需要说明的是，FANZA的榜单是销量榜，而国内的两个榜单是高评榜，因此肯定存在一定差异。但我们之前也提到过，JavLibrary和JavDB的榜单在排名时都使用了贝叶斯公式或其变体，即评分的人数越多越有利于排名——而按照我们实际观察的结果，在JavLibrary和JavDB进入榜单的影片，基本都是想看人数或看过人数排名靠前的影片——这某种程度上也可以等同于「销量」。因此，中日两国的榜单是有一定可比性的。

通过对这几个榜单的分析，又再次验证了我之前在图书馆和某堂发文时提到的一些观点。

对日本市场：

  • 大部分市场份额被新人片（高端市场份额至少1/4）和大牌偶像肉片（高端市场份额至少1/2）所占据。
  • 剧情片不重要。只能在新人和大牌的夹击下艰难地夹缝生存（高端市场份额估计在1/10左右）。
  • 导演不重要。市场对导演是谁，拍的是什么根本无所谓。市场只认大牌，只认新人。
  • 从女优类型上来讲，日本市场一认偶像，一认巨乳。

对国内观众：

  • 相对肉片更喜欢看剧情片，对新人则不感兴趣。
  • 审美上偏向于IDEA POCKET之类时尚精致的路线，而IDEA POCKET在日本只是二流片商。
  • 相对巨乳更喜欢丝袜、美腿。
  • 因为注重剧情，所以导演榜上排名靠前的都是剧情片拍得不错的导演。
