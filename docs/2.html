第二届JAV金鸡儿百花柳奖🐥🏆💯🌸

2023/01/20 发表于 jinjier.art


特别说明


在AV新法的冲击下，2022年的JAV影片无论数量还是质量相比往年都有所下降。在这一大趋势下，我们也对本届评奖做出了一些调整。

熟悉中国电影金鸡奖的影迷可能知道，金鸡奖曾有一段与大众电影百花奖合并评选的历史，合称为金鸡百花奖。

借鉴这一思路，虽然JAV金鸡儿奖是一个艺术类奖项（正如本站域名所暗示的），但我们也将在这一届评奖中充分考虑大众影迷的意见：在入围名单、奖项产生等各流程上同时参考砖家（CACA）和影迷（大数据）的意见。

因此，本届JAV金鸡儿奖全称为「第二届JAV金鸡儿百花柳奖」。标题和文中的💯🌸代表大众影迷意见。


评选范围


评选范围是2022年发行的JAV有码影片。


奖项设置


在奖项设置上，我们取的是中国电影金鸡奖和大众电影百花奖的交集，仅保留了最受大众关注的奖项，而取消了大部分技术类的评奖。具体奖项设置如下：


  1. 最佳故事片
  2. 最佳导演
  3. 最佳女主角
  4. 最佳男主角
  5. 终身成就AV艺术家


评选流程


  1. 与各大电影节类似，首先确定入围大名单。在入围名单上我们将同时参考砖家和大众影迷的意见。
  2. 从大名单中选出各奖项的提名名单。提名名单均按获提名影片的发行日期排序。
  3. 从提名名单中选出各个奖项。奖项可并列，可空缺。


入围名单


入围名额共22部，即2022年22部优秀影片。分别来自以下4个渠道：

  1. CACA年度十佳。代表砖家意见。
  2. JavLibrary年度TOP4。代表中国LSP的审美。
  3. JavDB年度TOP4。代表中国新司机的审美。
  4. FANZA销量榜年度TOP4。代表日本大众审美。


年度十佳


CACA砖家小组看重影片技术性（尤其是视听语言）和艺术性的呈现。年度十佳评选标准为：

  1. 全面考察编、导、摄、录、美、剪、演等技术环节。不能有明显的技术上的瑕疵。
  2. 在技术之外，还有其他亮点：或立意高远，或以情动人，或有创新意识，能跳出普通AV的窠臼。

年度十佳排名不分先后，按影片发行日期排序：



01. FSDSS-365 田舎の夏休みはヤることが無いから都会で洗練された幼なじみを帰省中にキメセク漬けにして絶頂しまくり即アクメBODYに仕上げた 橋本ありな
发行日期：2022-02-10
时长：115分钟
制作商：FALENO
发行商：FALENO star
导演：えび
女优：橋本ありな(1996-12-15 T167 84C-56-83)
