2023/04/24 发表于 jinjier.art



SSIS-698 三上悠亜と新ありなと相沢みなみ
发行日期：2023-05-09
时长：170分钟
制作商：S1 NO.1 STYLE
导演：ZAMPA
女优：三上悠亞 新ありな 相沢みなみ
男优：鮫島 小田切ジュン えりぐち

近期最受关注的影片，可能就是在三上悠亚退役前，由她与新有菜、相泽南合演的这部SSIS-698。


由一则评论说起


以下评论转载自JavDB，作者是我的小老弟、CACA会员seres：

这是她进入联盟后的第2872天。

2871天前，她在第一轮第一顺位被MUTEKI选中。她是天之娇女，最佳新秀，她的第一个赛季便拿下了总冠军（2015年FANZA影片年榜销冠，TEK-067）。她也用MVP（2017年FANZA成人奖最优秀女优奖）证明过自己。她在赛场外建立了庞大的商业帝国。她的社交媒体粉丝数连创新高。她的影响力已经超越了联盟——甚至有评论开始把她称作GOAT。

但也有不同的声音。有人说她在场上散步，床上划水；有人说她爱耍大牌，是更衣室的母狮；甚至有人因为她长得黑胖，把她叫作猩猩。

She is Queen Yua. 她叫三上悠亚。

--

这是她进入联盟后的第2580天。

她是2018年的MVP。她那时还是悠亚的队友，后来她为了更大的合同去了一支烂队——那是一段被蹉跎的时光——她渐渐开始被赛场遗忘。她换了脸，又换了一支强队，她连名字也换了——据说那是在向贾巴尔和慈世平致敬。

她叫新有菜。

--

这是她进入联盟后的第2414天。

没错，她是首轮秀，但她不过是首轮13顺位而已。她的前两个赛季主要在板凳上度过，第二个赛季她甚至差点被球队裁掉。她练得很苦，一有上场机会她就玩命。她的表现越来越好，她在不被看好的情况下拿下了2019年的MVP。

她叫相泽南。

--

新有菜和相泽南的手机同时响了。

是三上悠亚的声音：这个赛季结束后我就退役。把你们的天赋带到S1来吧，我在SSIS-698等着你们。我要你们也品尝到总冠军的滋味——我们一起再拼一次！


有趣的评论，将JAV和NBA进行类比，并玩了一些只有两边的粉丝能看懂的梗。


JAV vs NBA


NBA与JAV确实有相似之处。如都是运动(Sports)、娱乐(Entertainment)、游戏(Game)：

  • Sports. 不必多说，都是强对抗性运动。
  • Entertainment. 参与者有快感，也能让看官产生快感。
  • Game. 除了游戏外，Game一词也蕴含策略性、竞争性，如博弈论(Game Theory)。


正如NBA的粉丝中既有爱好上场打球的实战派，也有光看不练的理论党，JAV的粉丝也可分为实战派和撸管党：

粉丝类别SportsEntertainmentGame
NBA实战派✅✅✅
NBA理论党❌✅✅
JAV实战派✅✅✅
JAV撸管党❌✅❌
  • 对NBA和JAV的实战派而言，他们能同时获得Sports、Entertainment、Game的多重体验。
  • 对NBA理论党，Sports是没有的，但能通过代入球队胜负、球员统计等获得Game的体验。
  • 对JAV撸管党，Sports同样没有，Game性也缺失——实战派可以宣称自己连续几个时辰连续几个回合大战，自信感、征服感油然而生，简直令人肃然起敬；而撸管党要是号称自己连撸几小时，多半只会让人觉得可笑，甚至被视作撸射(Loser)。撸管仅能获得Entertainment，而缺少了Sports和Game，这可能也是撸管容易产生空虚感的原因……

本文一开始引用的评论中将「FANZA影片年榜销冠」类比为「总冠军」。我们先不去管这种类比是否合理，姑且接受这一设定，然后来看看历年「总冠军」的情况。

SELECT number, icon_url, SUBSTR(name, 0, 40), note
FROM ranks
WHERE note LIKE '影片榜%全年' AND number = 1
ORDER BY date

谈谈比赛的观赏性

我个人觉得是PRED-482 > IPZZ-034 > SSIS-698（本文写作时三部影片均未发售，以上观点仅针对预览）。

对这种多人运动，观赏性涉及很多方面，如场景、灯光、服装，动作戏的设计、调度，摄影的构图、运镜、剪辑、调色等等。

从动作戏的设计来看，IPZZ-034是明显胜过SSIS-698的，有「三女叠罗汉」等更丰富的玩法；但两部影片在摄影、美术方面则是半斤八两。而PRED-482在摄影、剪辑乃至拍摄观念上展现出的水平是要胜过那两部影片的。

这其实基本反映的是导演的水平。这也是我在JAV导演排行榜中对山口导演评价不错的原因。当然句号导演也是位不错的新生代导演。而ZAMPA，与TAKE-D、赤井彗星等一起，被我视作业界审美和技术最末流的导演（仅指摄影方面的审美和技术，他们当然可能有其他优点）。

SSIS-698的三巨头合演，因为选择了ZAMPA这位导演而蒙上了一层阴影。这正如顶级的食材，却选择了一位蹩脚的大厨，暴殄天物。


观赏性之调度


在体育赛事多人运动的拍摄中，实际一直都是有一个明确的焦点的。如足球、篮球比赛，毫无疑问球是比赛的焦点，镜头最需关注的自然是球的走向。

电影中的群戏比体育运动的拍摄要复杂些，因为需要导演确定拍摄的方向与焦点。而大场面、尤其是复杂场景的调度，更是极度考验导演的能力。黑泽明、沟口健二、小林正树等都是此中高手。

我一直不是特别喜欢看AV中N巨头的模式，因为这种模式实在很难拍出美感。焦点应该在哪？1对多时或许可以跟随男优的鸡儿……那多对多呢？运镜、剪辑的内在逻辑在哪？

以下面这部入选N巨头TOP榜单的影片为例，荒井导演已经算是处理得不错了。但也仅限于有台本的「综艺」部分，一旦进入到多人大乱交，仍然显得混乱：

槽点实在太多。稍微懂点修图的，查看该图片的RGB直方图，就会发现它严重泛红。演出场景的装修和色调也是土到掉渣。另外不难看到镜头里有不同色温的灯光。三位内衣颜色的搭配也是丑到爆炸💥，选择了高饱和度的、在RGB色轮上比较靠近的三种颜色，搭配在一起的视觉效果极为俗气……

这两张图片说明了什么问题？说明了SSIS-698的拍摄制作团队审美极差。

我曾在导演思维看AV中提到过，通过几个镜头基本就可以判断一个导演的水平。其实甚至都不用这么复杂，通过影片的封面和剧照，基本就可以判断拍摄制作团队的水平——这是我的不传之秘。

这也是我在前面将该片导演ZAMPA归为业界审美最末流的原因。对一个「文艺工作者」而言，审美是最重要的，方向一旦错误，其他都是在做无用功。

更愿意把钱花在搞营销、搞噱头上，而不太重视拍摄制作本身，这也是S1的老问题了——归根结底，这是观众惯的，是市场的选择。


看了反面教材，我们再来看一个正面的例子，获得第一届JAV金鸡儿奖最佳动作片的IPX-697：


IPX-697 禁欲の果て、汗と絶頂汁まみれで交わりまくった3日間 相沢みなみ
导演：イナバール

无需多言，也不用去分析摄影、灯光、剪辑、调色等技术细节；封面、预览呈现出的美感不言自明。

这部影片属于イナバール导演开创的「禁欲の果て、汗と絶頂汁」系列。我们来看这个系列的几张剧照：


IPX-536剧照 桃乃木かな

系列开山作。大战结束后，女优身上扔满了白色的安全套，极为色气。



IPX-697剧照 相沢みなみ

系列第三部。注意与上图的对比，安全套变成了红绿两色。加入点缀色后，画面一下生动了起来。



IPX-743剧照 桜空もも

系列第六部。安全套又增加了一种颜色，现在是三原色了。

仅从这几个安全套颜色的变化，我们不难体会到イナバール导演在美术细节处的用心。他在这个系列拍摄的几部作品也得到了JavLibrary & JavDB TOP榜单的高度认可。这说明国内LSP的审美是在线的。

但这个系列在日本市场却不怎么卖座，也从未进入过FANZA销量榜年榜。大概是因为拍摄制作成本高，又不怎么叫座，这个系列已经停更很久了。能说什么呢，毕竟还是市场决定一切吧。
