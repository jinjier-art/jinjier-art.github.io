JavLibrary & JavDB TOP榜单分析

2023/01/20 发表于 jinjier.art


背景介绍


JavLibrary和JavDB都是JAV领域的顶流站点。两个网站都维护了一个高分影片榜：

  • JavLibrary叫评价最高的影片，共500部，可以看作是「JavLibrary TOP500」。
  • JavDB则更会整活，模仿IMDB和豆瓣的TOP250，直接就叫「JavDB TOP250」。

两个榜单的完整数据都可以使用我们的JAV SQL 查询器查到。

需要注意的是：

  1. 两个榜单均变化频繁。本文写作时取的都是2023年1月1日的数据。
  2. 两个榜单排序时用的都是贝叶斯公式或其变体，这也是IMDB和豆瓣等榜单排序的常见算法——即评分的高低和评分的人数都会参与到排序计算中。

接下来我们将对这两个榜单进行数据分析，首先是「JavLibrary TOP500」，然后是「JavDB TOP250」。


JavLibrary TOP500


  • JavLibrary建站时间超过十年，主要使用PHP技术构建。
  • 网站外观长年鲜有变化，也没有移动版网页或手机App，略有点落后于时代。
  • 因为历史长，所以很多杰出的AV评论家都曾在这个舞台上闪耀过。
  • 因为历史长，也像我们一直强调的，JavLibrary代表的是中国LSP的审美。


可以使用以下SQL语句在JAV SQL 查询器中查询「JavLibrary TOP500」：

SELECT number, icon_url, SUBSTR(name, 0, 40), date, note
FROM ranks
WHERE kind = 5;
执行SQL


影片数量-javlib


  • 影片总数：500
  • 共演总数：77

JavLibrary只收录有码影片，因此TOP榜单上的500部都是有码影片。其中有77部由多名女优共演。


年份排名-javlib



上图是「JavLibrary TOP500」按入榜影片数量排名前十的年份。从以上数据我们不难发现，2012-2018年是JavLibrary的黄金时期，这也正对应了我们前面说的「JavLibrary代表的是中国LSP的审美」。

为节省篇幅，年份完整排名可在附录中查看。
